# Cross-Cultural Ad Risk Auditor / 跨文化广告风险审阅器

一个用于广告出海、品牌全球化传播和跨文化创意审查的 Claude Skill 文档项目。

通俗地说，它是一个“跨文化广告翻车预警器”：在广告投放前，帮助用户检查文案、视觉、符号、手势、颜色、人物设定、价值表达、平台语境和监管风险，提前发现可能引发误读、冒犯或品牌争议的元素。

> 本项目由一份《跨文化广告“翻车”预警器｜网络品牌传播课 Skill 策划书》整理升级而来，保留原方案中“跨文化传播 + 广告创意 + Skill 工具化”的核心思路，并补充了 GitHub 发布所需的项目结构、风险评级标准、输出模板、验证规则和免责声明。

## 为什么需要这个 Skill？

中国品牌出海、跨境电商、海外社媒运营、国际广告投放越来越常见，但广告创意在跨文化语境中容易出现问题：

- slogan 翻译生硬或产生歧义
- 颜色、手势、动物、数字、服饰、食物等视觉元素被误读
- 对宗教、性别、族群、国家符号、历史记忆处理不当
- 在不同平台语境中被二次传播、截屏、恶搞或引发争议
- 涉及健康、美妆、金融、儿童、食品等高监管品类时存在合规风险

这个 Skill 的价值不是“替代专家”，而是提供一个可复用的早期审查框架。

## 适用人群

- 广告学、传播学、市场营销、新媒体课程学生
- 品牌方、广告公司、内容运营、跨境电商团队
- 需要做海外社媒、短视频、落地页、本地化文案的人
- 想训练 Claude/ChatGPT/Gemini 做结构化广告审查的人

## 快速使用

1. 打开 `SKILL.md`。
2. 将其中内容作为 Claude Skill 的说明文件，或复制到你的项目知识库中。
3. 使用 `templates/input-template.md` 填写广告信息。
4. 让模型按 `templates/output-report-template.md` 输出风险报告。
5. 对高风险结论进行本地市场、母语者、法律合规或平台政策复核。

## 重要提醒

- 本 Skill 只用于早期风险预警，不构成法律、宗教、文化或合规最终意见。
- 不同国家、地区、年龄、宗教、社群和平台语境可能差异很大。
- 对高风险市场、高监管品类、大规模投放，必须进行本地专家复核。
- 使用文化模型时，避免把群体标签化或刻板化。
